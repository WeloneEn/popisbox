# У попи — сайт автомастерской

Сайт автомастерской во Владивостоке с полной переработкой структуры и визуального стиля.

## Концепция

- Бренд: автомастерская «У попи»
- Адрес: г. Владивосток, ул. Луговая, д. 74
- Стиль: современный сервисный интерфейс с акцентом на прозрачность работ
- Интерактив: анимированная машина на странице услуг (движение, дым из выхлопа, тень, режим турбо)

## Страницы

- `index.html` — главная и ключевые преимущества
- `work.html` — услуги, кейсы и анимация машины
- `about.html` — подход и принципы мастерской
- `testimonials.html` — отзывы клиентов
- `contact.html` — контакты и запись
- `project-1.html`, `project-2.html`, `project-3.html` — подробные кейсы

## Локальный запуск

1. Откройте `index.html` в браузере.
2. Либо запустите локальный сервер:

```bash
python -m http.server 8080
```

Откройте `http://localhost:8080/`.
